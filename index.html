    ✨ 特徴¶

    このシステムでできること

    • 測定範囲: 30mm～2000mm（0.03m～2m）
    • 高精度: ±3%の測定精度
    • リアルタイム: 連続測定とライブ表示
    • Phyphoxアプリ: リアルタイムグラフ表示
    • Bluetooth接続: ワイヤレスでデータ転送
    • CSV出力: データの保存と分析が可能
    • 複数班対応: 班ごとのセンサー管理
    • 実験支援: 物理・数学・情報の学習
    • 簡単セットアップ: 組み立て5分で実験開始

    🛠️ 必要な機材¶

    項目 説明 数量
    M5StickC Plus2 メインコントローラー 1台
    ToF HAT 距離センサーモジュール 1個
    📱 スマートフォン Phyphoxアプリ用 1台

    重要な注意点

    ToF HAT（本体上部装着タイプ）を使用してください。
    ToF UNIT（ケーブル接続タイプ）とは異なります。

    🚀 クイックスタート¶

    5分で実験開始！

    1. ハードウェア組み立て ⚡ 

      M5StickC Plus2 + ToF HAT → 本体上部に装着

    2. ソフトウェア準備 📱

    3. スマホにPhyphoxアプリをインストール

    4. Arduino IDEでプログラム書き込み

    5. 実験開始 🎉

    6. 電源ON → センサー名を確認
    7. Phyphoxで接続 → リアルタイム測定開始！
